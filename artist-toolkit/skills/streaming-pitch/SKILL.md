# Artist Streaming Pitch

The method for writing editorial/streaming pitches for an independent music artist. This
skill carries the *how*. The artist's identity, voice, and signature markers come from the
active project — read them, don't restate them.

## Principle — method, not identity

This skill is artist-agnostic on purpose, so it can serve any music project. Before writing:

- Pull the artist's voice and identity markers from the active project's instructions and
  knowledge base.
- Lean on previously written pitches stored in the project's knowledge base to keep tone and
  positioning consistent.
- If no artist voice is defined, ask for it before writing.
- Tone: professional but authentic. Never over-explain.

## Inputs to gather first

- Track title and release date
- Lyrics (to ground the pitch in the actual content)
- Context: place in the album, theme, inspiration
- Target platform — Spotify / Apple Music / Amazon Music (the fields vary slightly per platform)

## Spotify for Artists structure

1. Up to 3 genres
2. Up to 2 song styles
3. Up to 2 music cultures
4. Up to 2 moods
5. Up to 2 instruments
6. Description (500 characters max including spaces — roughly 75-100 words, 3-5 sentences): the track's story, universe, inspirations. No over-explanation.
   Professional but authentic.

Apple Music and Amazon Music use a similar shape with minor field differences — adapt the
structured fields to the target platform, keep the description method identical.

## The options question (ask once, before recommending)

Before recommending genres / styles / cultures / moods / instruments, ask:
"Tu veux que je te propose des recommandations, ou tu préfères me donner d'abord les options
exactes disponibles sur la plateforme pour que je choisisse dans la liste ?"

This guarantees the values match what's actually selectable on the platform and avoids
back-and-forth with non-matching choices.

## Output

List the structured fields first, then the description as its own clearly labeled block,
copy-ready for the platform form.
